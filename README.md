Kanban Board
A simple drag-and-drop Kanban board built with HTML, CSS, and JavaScript.

What is this project?
A Kanban Board where you can manage your tasks by organizing them into three columns:

To Do — Tasks you need to start
In Progress — Tasks you are currently working on
Done — Tasks you have completed
Features

➕ Add new tasks with a title and description
🗑️ Delete tasks with a delete button
🖱️ Drag and drop tasks between columns
💾 Tasks are saved automatically — even after page refresh!
📊 Task count shown on each column

🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Vanilla)
- localStorage (to save tasks in the browser)
